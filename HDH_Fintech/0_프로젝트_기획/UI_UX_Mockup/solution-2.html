2. HDH 핵심 손익 구조

HDH 솔루션의 핵심은 어떤 시나리오에서도 이익이 발생하는 구조입니다. 챌린지 합격/탈락 여부와 관계없이 수학적으로 수익이 보장됩니다.

2.1 시나리오 A: 챌린지 합격 시 (16%)

$300 계좌 기준 - 보험금 계좌에서 손실이 발생하지만, 챌린지 합격으로 더 큰 수익 확보

16% 확률
시나리오 A
챌린지 합격 + 보험금 SL
보험금 계좌 -$1,935 (SL)
펀딩 계좌 +$2,400 (합격)
참가비 -$300
보험금 계좌 수수료 -$18
펀딩 계좌 수수료 -$20
+$127
순이익률 21.2%

계산 상세

  • 펀딩 수익: +$2,400 (펀딩 계좌 트레이딩 수익)
  • 보험금 손절: -$1,935 (SL 21.5p 도달)
  • 참가비: -$300
  • 보험금 계좌 수수료: -$18
  • 펀딩 계좌 수수료: -$20
  • 순이익: $2,400 - $1,935 - $300 - $18 - $20 = +$127

2.2 시나리오 B: 챌린지 탈락 시 (84%)

$300 계좌 기준 - 챌린지 탈락으로 참가비를 잃지만, 보험금 계좌 TP 수익으로 보전

84% 확률
시나리오 B
챌린지 탈락 + 보험금 TP
보험금 계좌 +$369 (TP)
참가비 -$300 (탈락)
보험금 계좌 수수료 -$18
+$51
순이익률 8.5%

계산 상세

  • 보험금 익절: +$369 (TP 4.1p 도달)
  • 참가비 손실: -$300
  • 보험금 계좌 수수료: -$18
  • 순이익: $369 - $300 - $18 = +$51

2.3 실제 운영 시 고려사항

실전 운영에서 알아야 할 점

  • 슬리피지/스프레드: 청산 시 지정가격보다 밀려서 체결되거나, 매도/매수 호가 간의 갭으로 인해 수익이 줄어들고 손실이 늘어나 순이익이 감소할 수 있습니다.
  • 안전 운영: 안전한 운영을 위해 익절(TP) 포인트를 축소하거나 손절(SL) 포인트를 확대할 경우 순이익이 감소합니다.
핵심 결론
시나리오 A - 합격 (16%): +$127 순이익 (순이익률 21.2%)
시나리오 B - 탈락 (84%): +$51 순이익 (순이익률 8.5%)
